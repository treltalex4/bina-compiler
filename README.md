# bina-compiler

Компилятор для языка **Bina** — статически типизированного императивного языка с явной мутабельностью. Реализован на C++23.

## Сборка

```bash
cmake --preset default
cmake --build build
```

Требуется: `clang++-22`, `clang-scan-deps-22`, `libc++-22`, CMake ≥ 3.30, Ninja.

## Запуск

```bash
./build/bina <source.bina>              # компиляция
./build/bina --dump-tokens <source>     # вывод потока токенов
./build/bina --dump-ast <source>        # вывод AST
```

Примеры программ — в [examples/](examples/).

## Дополнительные возможности

- **C++23 modules** — компилятор реализован с использованием модулей C++23 (`import std;`, `.cppm`) вместо классических заголовочных файлов.
- **Накопление ошибок** — парсер не останавливается на первой ошибке, а собирает все диагностики через механизм синхронизации.
- **Методы структур (`impl`)** — поддержка методов через блоки `impl`, вызов через точку (`obj.method()`) с автоподстановкой `self`.
- **Вывод типов** — указание типа в `let`/`mut` необязательно: `let x = 42;` (планируется).
- **Неявное приведение типов** — автоматическое расширение числовых типов в выражениях (планируется).
- **Перегрузка функций** — несколько функций с одним именем, различающихся сигнатурой (планируется).
